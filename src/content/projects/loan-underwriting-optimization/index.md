---
title: "Loan Underwriting Optimization"
description: "A logistic-regression project for predicting one-year probability of default and improving underwriting decisions."
date: "2025-05-01"
---

This project explored a bank loan underwriting workflow using a logistic regression model to predict one-year probability of default for prospective borrowers.

I used historical bank transaction data to engineer critical financial features, including liquidity, debt coverage, profitability, and leverage. I then applied feature selection and validation analysis to reduce multicollinearity and improve model reliability.

The final model achieved an AUC of 0.7761, outperforming a baseline model with an AUC of 0.701.
